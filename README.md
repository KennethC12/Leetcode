## 🛠 How to Use
1. Clone the repository:
   ```sh
   git clone https://github.com/KennethC12/Leetcode.git
   ```
2. Navigate into the directory:
   ```sh
   cd LeetCode
   ```
3. Run Python scripts:
   ```sh
   python3 filename.py
   ```
